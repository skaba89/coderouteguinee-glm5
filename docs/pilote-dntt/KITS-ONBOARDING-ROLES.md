## 1. Kit Candidat — "Mon parcours permis de conduire"

### 1.1 Bienvenue

Bienvenue sur **CodeRoute Guinée**, la plateforme officielle de la DNTT pour passer votre permis de conduire.

Ce document vous guide pas à pas, de votre inscription à la réception de vos résultats.

### 1.2 Étape 1 — Inscription (10 min)

1. Rendez-vous sur `https://coderoute.gov.gn`
2. Cliquez sur "Se connecter" puis "Créer un compte"
3. Remplissez le formulaire :
   - Nom et prénom (tels que sur votre pièce d'identité)
   - Date de naissance
   - Numéro de pièce d'identité (carte d'identité guinéenne ou passeport)
   - Téléphone (format +224 6XX XX XX XX)
   - Ville (Conakry, Kankan, Labé — selon le centre pilote)
   - Email (si vous en avez un)
   - Mot de passe (8 caractères min, 1 majuscule, 1 chiffre, 1 caractère spécial)
4. Choisissez votre langue : Français, Pular, Soussou ou Malinké
5. Acceptez les conditions d'utilisation et la politique de confidentialité
6. Validez. Vous recevez un SMS de confirmation avec votre numéro unique (format `GN-CODE-YYYY-XXXXXX`)

**Conseil** : gardez votre numéro unique précieusement, il vous servira pour toutes vos démarches.

### 1.3 Étape 2 — Révision (au votre rythme)

Une fois connecté, accédez à votre tableau de bord :

- **Cours** : modules théoriques (panneaux, règles, sécurité)
- **Entraînement** : quiz thématiques (10-20 questions chacun)
- **Examens blancs** : simulations complètes (40 questions, 30 min)

**Recommandation** : complétez au moins 5 quiz avant de réserver votre examen.

### 1.4 Étape 3 — Réservation examen (5 min)

1. Cliquez sur "Réserver"
2. Choisissez votre centre (Conakry-Kaloum, Kankan ou Labé)
3. Sélectionnez un créneau disponible
4. Vérifiez le montant (35 000 GNF pour examen code, 50 000 GNF pour conduite)
5. Choisissez votre moyen de paiement : Orange Money ou MTN MoMo
6. Saisissez votre numéro de téléphone Mobile Money
7. Validez. Vous recevez une demande de paiement sur votre téléphone.
8. Confirmez le paiement avec votre code secret Orange/MTN.
9. Vous recevez votre confirmation par SMS + email avec votre numéro de convocation.

### 1.5 Étape 4 — Jour de l'examen

1. Arrivez 30 minutes avant l'heure prévue au centre
2. Présentez votre convocation (SMS ou imprimée) + pièce d'identité
3. Suivez les instructions de l'agent du centre
4. Examen code : 40 questions en 30 minutes, sur tablette ou ordinateur
5. Examen conduite : 30 minutes en véhicule du centre
6. À la fin, votre score s'affiche. Vous le recevez aussi par SMS et email.

### 1.6 Étape 5 — Résultats

- Si réussite : vous pouvez réserver l'examen suivant (conduite si vous avez réussi le code)
- Si échec : vous pouvez réserver une nouvelle session après 7 jours
- Vos résultats sont archivés dans votre tableau de bord "Résultats"

### 1.7 Support

- **WhatsApp** : +224 6XX XX XX XX (8h-20h, 7j/7)
- **Email** : support@coderoute.gov.gn
- **Téléphone** : +224 6XX XX XX XX (8h-20h, lun-ven)
- **FAQ** : disponible dans l'application menu "Aide"

### 1.8 Conseils

- **Révisez régulièrement** : 30 min/jour vaut mieux que 5h la veille
- **Entraînez-vous aux examens blancs** : ils simulent les conditions réelles
- **Vérifiez votre convocation** : date, heure, centre
- **Arrivez à l'heure** : 30 min avant l'heure prévue
- **Pas de téléphone pendant l'examen** : c'est de la fraude, vous serez disqualifié
- **En cas de problème** : contactez le support AVANT l'examen, pas après

### 1.9 Vos droits RGPD

Conformément à la Loi L/2022/018/AN, vous avez droit à :
- **Accès** à vos données personnelles
- **Rectification** si elles sont incorrectes
- **Effacement** (sauf obligation légale de conservation)
- **Opposition** au traitement
- **Portabilité** de vos données

Pour exercer ces droits : écrivez à `rgpd@coderoute.gov.gn` avec votre numéro unique.

## 2. Kit Auto-École — "Former mes élèves avec CodeRoute"

### 2.1 Votre rôle

En tant qu'auto-école partenaire du pilote DNTT, vous :
- Insérez vos élèves dans CodeRoute (création comptes candidats)
- Suivez leur progression (cours, quiz, examens)
- Payez les frais d'examen pour le compte de vos élèves
- Accédez à des statistiques de réussite par élève

### 2.2 Connexion

1. Rendez-vous sur `https://coderoute.gov.gn`
2. Cliquez "Se connecter"
3. Saisissez votre email auto-école + mot de passe
4. Saisissez le code 2FA (application Google Authenticator)
5. Vous êtes sur votre tableau de bord auto-école

### 2.3 Inscrire un élève

1. Menu "Élèves" → "Ajouter un élève"
2. Remplissez le formulaire (nom, prénom, identité, téléphone, ville)
3. L'élève reçoit un SMS avec son numéro unique + mot de passe temporaire
4. L'élève doit changer son mot de passe à la première connexion

**Important** : ne créez pas de compte sans le consentement explicite de l'élève (RGPD).

### 2.4 Suivre un élève

- Menu "Élèves" → sélectionnez l'élève
- Visualisez : cours complétés, quiz passés, scores, examens à venir
- Téléchargez un rapport PDF de progression

### 2.5 Réserver un examen pour un élève

1. Menu "Élèves" → sélectionnez l'élève → "Réserver un examen"
2. Choisissez centre + créneau
3. Paiement : soit l'élève paie lui-même, soit l'auto-école paie (Mobile Money)
4. Confirmation par SMS à l'élève + email à l'auto-école

### 2.6 Statistiques

- Menu "Statistiques" → vue d'ensemble
- Filtres : période, élève, type examen
- Indicateurs : taux réussite, panier moyen, nombre d'examens
- Export CSV / PDF

### 2.7 Gestion compte

- Menu "Mon compte" → modifier profil, mot de passe, 2FA
- Menu "Utilisateurs" → gérer les comptes de vos collaborateurs (mono-poste ou multi-poste)
- Menu "Facturation" → historique des paiements, factures téléchargeables

### 2.8 Support auto-école

- **WhatsApp pro** : +224 6XX XX XX XX (8h-20h)
- **Email dédié** : auto-ecoles@coderoute.gov.gn
- **Formation continue** : webinaires mensuels (annonce par email)
- **Documentation** : `docs/pilote-dntt/CONVENTION-CENTRE-PILOTE.md`

### 2.9 Bonnes pratiques

- **Vérifiez l'identité** de chaque élève avant création de compte
- **Suivez la progression** hebdomadairement, pas juste avant l'examen
- **Payez en avance** (J-7 minimum) pour réserver les meilleurs créneaux
- **Communiquez les convocations** à vos élèves (SMS + email)
- **Anonymisez** les données des élèves ayant quitté votre auto-école (RGPD)
- **Sauvegardez** vos rapports de progression (téléchargement mensuel)

## 3. Kit Centre Agréé — "Organiser les examens"

### 3.1 Votre rôle

En tant que centre agréé pilote (Conakry-Kaloum, Kankan, Labé), vous :
- Organisez les sessions d'examen (créneaux, salles, surveillants)
- Accueillez les candidats le jour J
- Supervisez le déroulement des examens
- Remontez les résultats dans la plateforme
- Signalez les incidents (fraude, panne, comportement suspect)

### 3.2 Connexion

1. `https://coderoute.gov.gn` → "Se connecter"
2. Email centre + mot de passe + 2FA
3. Tableau de bord centre s'affiche

### 3.3 Gérer le planning

- Menu "Planning" → vue calendrier mensuel
- Créez des créneaux : date, heure, type examen (code/conduite), capacité
- Les candidats voient vos créneaux en temps réel
- **Anticipez** : publiez les créneaux 14 jours à l'avance minimum
- **Capacité maximale** : conforme à votre agrément (Conakry 200/jour, Kankan 80, Labé 50)

### 3.4 Jour d'examen

#### 30 min avant

1. Vérifiez la salle (propreté, éclairage, ventilation)
2. Testez le matériel : tablettes/ordinateurs, connexion Internet, projecteur (si affichage règles)
3. Imprimez la liste des candidats attendus (menu "Sessions" → "Liste émargement")
4. Préparez les convocations et pièces d'identité à vérifier

#### Accueil des candidats

1. Vérifiez l'identité de chaque candidat (pièce d'identité ↔ convocation)
2. Faites émarger la liste
3. Dirigez le candidat vers sa place
4. Rappel des règles : pas de téléphone, pas de discussion, pas de triche

#### Pendant l'examen

1. Surveillez activement (1 surveillant pour 20 candidats max)
2. En cas de fraude avérée :
   - Suspendez la session du candidat (bouton "Suspendre" dans la plateforme)
   - Confisque le matériel de fraude
   - Rédige un rapport d'incident (menu "Incidents" → "Nouveau")
   - Notifie le RSSI via Slack `#securite-incident`
3. En cas de panne technique :
   - Si < 5 min : reprenez l'examen
   - Si > 5 min : annulez la session, reprogrammez gratuitement
   - Documentez l'incident

#### Après l'examen

1. Les résultats sont calculés automatiquement par la plateforme
2. Vérifiez les résultats (pas de score aberrant)
3. Validez les résultats (menu "Sessions" → "Valider résultats")
4. Les candidats reçoivent automatiquement SMS + email
5. Archivenz la liste émargement (5 ans, obligation légale)

### 3.5 Statistiques centre

- Menu "Statistiques"
- Indicateurs : taux remplissage, taux réussite, taux fraude, taux incident
- Comparaison vs autres centres pilotes (anonymisé)
- Export mensuel pour rapport DNTT

### 3.6 Gestion staff centre

- Menu "Staff" → ajouter/supprimer surveillants
- Chaque surveillant a un compte dédié (pas de compte partagé)
- 2FA obligatoire pour tous les comptes staff
- Audit log : toutes les actions sont tracées

### 3.7 Support centre

- **WhatsApp groupe** centres pilotes (réponse < 30 min)
- **Email** : centres@coderoute.gov.gn
- **Téléphone urgence** : +224 6XX XX XX XX (24/7 pendant pilote)
- **Visio hebdo** : mercredi 11h00 (point entre centres)
- **Documentation** : `docs/pilote-dntt/CONVENTION-CENTRE-PILOTE.md`

### 3.8 Conformité RGPD centre

- **Ne communiquez pas** les données candidats à des tiers
- **Ne photocopiez pas** les pièces d'identité (sauf obligation légale)
- **Anonymisez** les listes émargement après 5 ans
- **Signalez** tout incident RGPD au RSSI sous 24h
- **Formation** : tous les surveillants doivent suivre la formation RGPD (1h)

## 4. Kit Administration DNTT — "Piloter le système"

### 4.1 Votre rôle

En tant qu'agent DNTT (administration), vous :
- Avez une vue d'ensemble du pilote
- Gérez les centres agréés (agrément, suspension, révocation)
- Gérez les auto-écoles (validation, suspension)
- Gérez les candidats (support niveau 2, escalades)
- Supervisez la sécurité (anti-fraude, audit log)
- Générez des rapports pour la direction DNTT

### 4.2 Connexion sécurisée

1. `https://coderoute.gov.gn` → "Se connecter"
2. Email administration + mot de passe + 2FA TOTP
3. **Obligatoire** : connexion depuis le réseau DNTT (VPN ou IP fixe)
4. **Astreinte** : 1 admin d'astreinte 24/7 pendant pilote

### 4.3 Vue d'ensemble

- Menu "Vue d'ensemble" → dashboard temps réel
- Indicateurs : inscriptions cumul, examens/jour, paiements/jour, incidents, uptime
- Filtres : période, centre, type examen
- Alertes actives affichées en haut

### 4.4 Gestion centres agréés

- Menu "Centres"
- Actions : créer, modifier, suspendre, révoquer
- Vérification : agrément en cours, capacité, équipements
- Statistiques par centre (taux réussite, fraude, incidents)
- **Procédure suspension** : 3 avertissements → suspension 7 jours → révocation

### 4.5 Gestion auto-écoles

- Menu "Auto-écoles"
- Validation des nouvelles auto-écoles (vérification juridique)
- Suspension en cas de fraude ou non-respect convention
- Statistiques par auto-école (élèves, examens, taux réussite)

### 4.6 Gestion candidats

- Menu "Candidats"
- Recherche par numéro unique, nom, téléphone
- Actions : consulter profil, réinitialiser mot de passe, bloquer compte (fraude), supprimer compte (RGPD)
- Historique complet : inscriptions, paiements, examens, alertes fraude
- **Procédure blocage** : motif obligatoire + validation hiérarchique (super-admin)

### 4.7 Anti-fraude

- Menu "Anti-fraude"
- Alertes temps réel par sévérité (info, low, medium, high, critical)
- Actions : investigation, blocage candidat, blocage IP, escalation RSSI
- Outils : heat map IP, patterns réponse, détection identité suspecte
- **Procédure critique** : investigation < 1h, décision < 4h, communication < 24h

### 4.8 Audit log

- Menu "Audit" (réservé super-admin + RSSI)
- Toutes les actions sensibles sont tracées
- Filtres : utilisateur, type action, période
- Export pour audit externe ou enquête
- Conservation : 5 ans (art. 35 Loi L/2022/018/AN)

### 4.9 Communications

- Menu "Communications"
- Envoi emails / SMS à des segments (tous candidats, candidats Conakry, etc.)
- Templates multilingues pré-définis
- Historique des communications envoyées
- **Validation** : les communications de masse nécessitent validation hiérarchique

### 4.10 Statistiques & rapports

- Menu "Analyses"
- Tableaux de bord avancés (par centre, par période, par catégorie permis)
- Rapports planifiés : hebdo (lundi 09h), mensuel (1er lundi mois)
- Export PDF / Excel / CSV
- API d'extraction pour outils BI externes (sur demande)

### 4.11 Configuration système

- Menu "Configuration" (réservé super-admin)
- Gestion tarifs (TarifConfig)
- Gestion quotas (par centre, par jour)
- Gestion IPs autorisées / bannies
- Gestion variables d'environnement (sensibles)

### 4.12 Support niveau 2

- Menu "Support"
- Tickets escaladés par support L1
- SLA : réponse < 4h ouvrées, résolution < 48h
- Catégories : technique, paiement, fraude, RGPD, autre
- Escalade super-admin si besoin

### 4.13 Astreinte administration

- 1 admin d'astreinte 24/7 pendant pilote
- Rotation semaison (lundi 09h à lundi suivant 09h)
- Téléphone astreinte : +224 6XX XX XX XX
- Intervention : < 1h pour critique, < 4h pour élevé
- Procédure escalade : voir `RUNBOOK-GO-LIVE.md` §5.2

### 4.14 Documentation admin

- `docs/pilote-dntt/TABLEAU-DE-BORD-KPI.md` — KPI à surveiller
- `docs/pilote-dntt/CALENDRIER-PILOTE-8SEM.md` — planning global
- `docs/ops/OPS-RUNBOOK.md` — procédures incident
- `docs/gouvernance/COMITE-PILOTAGE-SECURITE.md` — gouvernance hebdo
