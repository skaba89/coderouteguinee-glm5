## 1. Inscription & Compte

### FR — Questions fréquentes

**Q1 : Comment m'inscrire sur CodeRoute ?**
R : Rendez-vous sur `https://coderoute.gov.gn`, cliquez "Se connecter" puis "Créer un compte". Remplissez le formulaire (nom, prénom, identité, téléphone, ville, mot de passe). Vous recevez un SMS avec votre numéro unique `GN-CODE-YYYY-XXXXXX`.

**Q2 : Je n'ai pas reçu le SMS de confirmation.**
R : Vérifiez votre numéro de téléphone (format +224). Attendez 5 minutes maximum. Si toujours rien, contactez le support WhatsApp +224 6XX XX XX XX.

**Q3 : J'ai oublié mon mot de passe.**
R : Cliquez "Mot de passe oublié" sur la page de connexion. Saisissez votre numéro unique ou email. Vous recevez un lien de réinitialisation par SMS et email.

**Q4 : Puis-je avoir plusieurs comptes ?**
R : Non. Un numéro d'identité = un compte. La création de comptes multiples est de la fraude et entraîne la fermeture immédiate de tous vos comptes.

**Q5 : Mes données personnelles sont-elles sécurisées ?**
R : Oui. CodeRoute est conforme à la Loi L/2022/018/AN (RGPD guinéenne). Vos données sont chiffrées, stockées en Guinée, et ne sont jamais partagées avec des tiers sans votre consentement. Vous pouvez exercer vos droits (accès, rectification, effacement) à `rgpd@coderoute.gov.gn`.

**Q6 : Puis-je changer de langue après inscription ?**
R : Oui. Menu "Mon profil" → "Langue". Choix : Français, Pular, Soussou, Malinké.

## 2. Cours & Révision

### FR

**Q7 : Combien de cours dois-je suivre avant l'examen ?**
R : Il n'y a pas de minimum obligatoire, mais nous recommandons de compléter au moins 80% des modules + 5 quiz thématiques avant de réserver l'examen code.

**Q8 : Les quiz sont-ils chronométrés ?**
R : Les quiz thématiques ne sont pas chronométrés (vous prenez le temps qu'il faut). Les examens blancs sont chronométrés (30 minutes, comme l'examen réel).

**Q9 : Puis-je revoir mes réponses après un quiz ?**
R : Oui. Menu "Entraînement" → "Historique" → sélectionnez le quiz. Vous voyez vos réponses + les bonnes réponses + explications.

**Q10 : Les cours sont-ils disponibles hors-ligne ?**
R : Pas encore pour le pilote. Prévu pour la généralisation (S9+). En attendant, téléchargez le PDF de chaque module (bouton "Télécharger PDF").

**Q11 : Existe-t-il des vidéos ?**
R : Oui, certains modules ont des vidéos (signalées par une icône ▶). Durée : 2-5 min chacune.

**Q12 : Les contenus sont-ils disponibles en Pular/Soussou/Malinké ?**
R : Les cours sont en Français principalement. Les vidéos ont des sous-titres dans les 4 langues. Les quiz peuvent être passés en 4 langues. L'interface est disponible en 4 langues.

## 3. Réservation Examen

### FR

**Q13 : Comment réserver un examen ?**
R : Menu "Réserver" → choisissez centre + créneau + type examen → payez 35 000 GNF (code) ou 50 000 GNF (conduite) via Orange Money ou MTN MoMo. Confirmation immédiate par SMS + email.

**Q14 : Quels centres pilotes sont disponibles ?**
R : 3 centres pour le pilote : Conakry-Kaloum (capacité 200/jour), Kankan (80/jour), Labé (50/jour).

**Q15 : Combien de temps à l'avance dois-je réserver ?**
R : Minimum 48h à l'avance. Recommandation : 7 jours pour avoir le choix des créneaux.

**Q16 : Puis-je modifier ma réservation ?**
R : Oui, jusqu'à 48h avant l'examen, sans frais. Moins de 48h : 50% de frais. Moins de 24h ou absent : pas de remboursement.

**Q17 : Que faire si je rate mon examen ?**
R : Vous pouvez réserver une nouvelle session après 7 jours. Le paiement d'un nouvel examen est obligatoire (35 000 GNF code, 50 000 GNF conduite).

**Q18 : Puis-je passer l'examen conduite sans avoir réussi le code ?**
R : Non. Vous devez obligatoirement réussir l'examen code avant de réserver la conduite.

**Q19 : Le centre le plus proche est complet. Que faire ?**
R : Choisissez un autre centre pilote, ou attendez que de nouveaux créneaux soient publiés (généralement chaque lundi).

## 4. Paiement

### FR

**Q20 : Quels moyens de paiement sont acceptés ?**
R : Orange Money et MTN MoMo uniquement pour le pilote. Le cash et la carte bancaire ne sont pas acceptés à distance (uniquement sur place dans certains centres, à terme).

**Q21 : Le paiement a échoué mais j'ai été débité.**
R : Contactez immédiatement le support WhatsApp avec votre numéro de transaction Orange/MTN. Nous vérifions et remboursons sous 48h si l'échec est confirmé côté plateforme.

**Q22 : Combien coûte l'examen ?**
R : Examen code : 35 000 GNF. Examen conduite : 50 000 GNF. Tarifs identiques dans tous les centres pilotes. Tarifs susceptibles d'évoluer après pilote (avec validation DNTT).

**Q23 : Puis-je obtenir un remboursement ?**
R : Oui, dans les cas suivants :
- Annulation > 48h avant : 100% remboursé
- Annulation 24-48h avant : 50% remboursé
- Annulation < 24h ou absence : pas de remboursement
- Annulation par la plateforme (panne) : 100% remboursé ou reprogrammation gratuite
- Décès du candidat : 100% remboursé (sur présentation acte de décès)

**Q24 : Comment obtenir une facture ?**
R : Menu "Mes paiements" → sélectionnez la transaction → "Télécharger facture PDF". Format légal guinéen, mention TVA.

**Q25 : Ma femme/mari peut payer pour moi ?**
R : Oui. Saisissez le numéro Mobile Money de la personne qui paie (pas forcément le vôtre). La transaction reste liée à votre compte candidat.

## 5. Jour de l'Examen

### FR

**Q26 : Que dois-je apporter le jour J ?**
R : 3 documents obligatoires :
1. Pièce d'identité originale (carte d'identité guinéenne ou passeport)
2. Convocation (SMS sur votre téléphone ou imprimée)
3. Numéro unique CodeRoute (`GN-CODE-YYYY-XXXXXX`)

**Q27 : Combien de temps dure l'examen code ?**
R : 30 minutes pour 40 questions. Pas de pause possible. Sortie définitive en cas de sortie de la salle.

**Q28 : Que faire si je suis en retard ?**
R : Vous avez 15 minutes de tolérance. Au-delà, l'accès à l'examen est refusé (considéré comme absent). Reprogrammez une nouvelle session (paiement obligatoire).

**Q29 : Puis-je utiliser ma calculatrice ?**
R : Non. Aucun appareil électronique autorisé (téléphone, montre connectée, calculatrice, écouteurs). Casque audio fourni si nécessaire pour les versions audio.

**Q30 : Que se passe-t-il en cas de triche ?**
R : Sanctions immédiates :
- Examen annulé (note 0)
- Compte suspendu 6 mois
- Possibilité de poursuites pénales (fraude documentaire)
- Inscription au registre des fraudeurs DNTT

**Q31 : Puis-je passer l'examen en Pular/Soussou/Malinké ?**
R : Oui. À la réservation, choisissez votre langue. Le jour J, l'interface et les questions seront dans cette langue. Surveillance bilingue (français + langue choisie) dans la mesure du possible.

**Q32 : En cas de panne technique pendant l'examen ?**
R : Si panne < 5 min : reprise de l'examen là où il s'est arrêté. Si panne > 5 min : annulation et reprogrammation gratuite sous 7 jours. Aucun impact sur votre dossier.

## 6. Résultats

### FR

**Q33 : Quand reçois-je mes résultats ?**
R : Immédiatement à la fin de l'examen (affiché à l'écran). Vous recevez aussi un SMS et un email dans les 5 minutes. Les résultats sont visibles dans votre tableau de bord "Résultats".

**Q34 : Quelle est la note de réussite ?**
R : Code : 35/40 (87.5%) minimum. Conduite : évaluation par l'inspecteur sur 40 points, 28/40 (70%) minimum.

**Q35 : Puis-je contester mes résultats ?**
R : Oui, sous 48h, en écrivant à `contestation@coderoute.gov.gn` avec votre numéro unique + date examen + motif. Une commission DNTT examine sous 7 jours. Décision définitive.

**Q36 : Combien de temps sont conservés mes résultats ?**
R : 10 ans (obligation légale). Vous pouvez les consulter et télécharger à tout moment.

**Q37 : Puis-je obtenir une attestation de réussite ?**
R : Oui. Menu "Résultats" → "Télécharger attestation". Document officiel DNTT avec QR code de vérification.

## 7. Technique

### FR

**Q38 : La plateforme ne s'ouvre pas.**
R : 1) Vérifiez votre connexion Internet. 2) Essayez un autre navigateur (Chrome, Firefox, Safari recommandés). 3) Videz le cache navigateur. 4) Si toujours KO, contactez support WhatsApp.

**Q39 : L'application est lente.**
R : Vérifiez votre connexion (3G minimum recommandée). Fermez les autres onglets. Essayez en hors-pointe (avant 9h, après 22h). Si toujours lent, signalez-le au support.

**Q40 : Puis-je utiliser CodeRoute sur mon téléphone ?**
R : Oui, la plateforme est responsive (s'adapte à toutes tailles d'écran). Application mobile PWA installable (Android + iPhone) via le menu navigateur "Installer l'application".

**Q41 : Quels navigateurs sont supportés ?**
R : Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. Internet Explorer non supporté.

**Q42 : Mes données sont-elles sauvegardées ?**
R : Oui, sauvegarde automatique toutes les 30 secondes pendant les quiz. Vous pouvez reprendre où vous étiez en cas de coupure.

## 8. RGPD & Confidentialité

### FR

**Q43 : Quelles données personnelles sont collectées ?**
R : Nom, prénom, date de naissance, numéro d'identité, téléphone, email (optionnel), ville, photo (optionnelle), résultats d'examens, historique de navigation sur la plateforme.

**Q44 : Où sont stockées mes données ?**
R : En Guinée, sur des serveurs sécurisés du datacenter DNTT Conakry (avec réplication Kankan pour la continuité). Jamais à l'étranger.

**Q45 : Qui a accès à mes données ?**
R : Accès strictement limité :
- Vous-même
- Agents DNTT autorisés (avec audit log)
- Votre auto-école (si inscrite via auto-école)
- Le centre où vous passez l'examen (le jour J uniquement)
- L'auditeur externe (sous NDA, données anonymisées)
- L'AGPD sur réquisition légale

**Q46 : Comment exercer mes droits RGPD ?**
R : Écrivez à `rgpd@coderoute.gov.gn` avec votre numéro unique + pièce d'identité + demande. Réponse sous 30 jours. Droits : accès, rectification, effacement, opposition, portabilité.

**Q47 : Puis-je supprimer mon compte ?**
R : Oui. Menu "Mon profil" → "Supprimer mon compte". Attention : les résultats d'examens sont conservés 10 ans (obligation légale). Le reste est supprimé sous 30 jours.

**Q48 : Que faire en cas de fuite de données ?**
R : CodeRoute s'engage à vous notifier sous 72h en cas de violation de vos données (article 33 Loi L/2022/018/AN). Si vous constatez une anomalie, signalez-la immédiatement à `securite@coderoute.gov.gn`.

## 9. Auto-Écoles

### FR

**Q49 : Comment mon auto-école peut-elle devenir partenaire ?**
R : Pendant le pilote, sur invitation DNTT uniquement. Après généralisation : dossier de candidature via `auto-ecoles@coderoute.gov.gn` (agrément en cours obligatoire).

**Q50 : Combien d'élèves puis-je inscrire ?**
R : Pas de limite pour le pilote. Recommandation : commencez par 10-20 élèves pour valider votre processus.

**Q51 : Puis-je payer pour mes élèves ?**
R : Oui. Menu "Élèves" → "Payer pour un élève". Saisissez votre numéro Mobile Money (auto-école). Facture globale disponible en fin de mois.

**Q52 : Comment suivre la progression de mes élèves ?**
R : Menu "Élèves" → sélectionnez l'élève. Vue complète : cours, quiz, scores, examens. Export PDF mensuel possible.

## 10. Centres Agréés

### FR

**Q53 : Comment devenir centre pilote ?**
R : Sur sélection DNTT. Critères : agrément en cours, capacité d'accueil (≥ 50/jour), connexion Internet stable, équipe formée, engagement 8 semaines.

**Q54 : Combien d'examens puis-je organiser par jour ?**
R : Selon votre capacité agrée : Conakry-Kaloum 200/jour, Kankan 80/jour, Labé 50/jour. Modalités : réparties en 2-3 sessions de 2h.

**Q55 : Que faire en cas d'incident pendant une session ?**
R : 1) Suspendez la session (bouton "Suspendre"). 2) Notifiez le support WhatsApp centres. 3) Rédigez un rapport d'incident (menu "Incidents"). 4) Reprogrammez les candidats concernés gratuitement.

**Q56 : Comment sont rémunérés les centres ?**
R : Pendant le pilote : pas de rémunération directe (gratuité du pilote). Après généralisation : commission sur chaque examen organisé (à définir par convention).

## 12. Après le Pilote

### FR

**Q61 : Que se passe-t-il après le pilote (8 semaines) ?**
R : Bilan officiel présenté au Ministère des Transports. Si GO : généralisation à tout le territoire (extension centres, plus de candidats). Si NO-GO : extension pilote ou repense du système.

**Q62 : Mes résultats d'examen seront-ils valides après le pilote ?**
R : Oui. Tous les examens passés pendant le pilote sont officiellement valables (équivalent permis de conduire guinéen).

**Q63 : Vais-je devoir payer à nouveau après le pilote ?**
R : Non pour les examens déjà passés. Si vous devez passer d'autres examens après le pilote, les tarifs applicables seront ceux en vigueur à ce moment (potentiellement ajustés).

**Q64 : Quand la plateforme sera-t-elle disponible dans tout le pays ?**
R : Décision officielle après le pilote. Estimation : 6 mois après GO pour la généralisation (16 centres potentiels).
